# Создание смет

**Версия:** 2.3.0

## Способы создания

### 1. С нуля (Ctrl+N)

```
1. Нажать "Новая смета" или Ctrl+N
2. Заполнить данные клиента (имя, телефон, email)
3. Указать PAX и даты
4. Добавить услуги
5. Настроить наценки
6. Сохранить (Ctrl+S)
```

### 2. Из каталога

```
1. Открыть каталог (Ctrl+K)
2. Выбрать готовый пакет
3. Система создаст смету с услугами
4. Скорректировать данные клиента
5. Сохранить
```

### 3. Дублирование

```
1. Открыть существующую смету
2. Меню → Дублировать (или Ctrl+D)
3. Изменить данные клиента
4. Скорректировать услуги/цены
5. Сохранить под новым именем
```

### 4. Импорт из файла

```
1. Меню → Импорт
2. Выбрать JSON файл
3. Система валидирует данные
4. Смета создаётся автоматически
5. Проверить и сохранить
```

## Шаблон пустой сметы

```json
{
  "version": "1.1.0",
  "metadata": {
    "created": "2025-11-06T07:00:00Z",
    "modified": "2025-11-06T07:00:00Z"
  },
  "client": {
    "name": "",
    "phone": "",
    "email": "",
    "notes": ""
  },
  "group": {
    "pax": 1,
    "startDate": "",
    "endDate": "",
    "duration": { "days": 0, "nights": 0 }
  },
  "services": [],
  "pricing": {
    "hiddenMarkup": 0,
    "taxRate": 0
  }
}
```

## Быстрое создание

**Для повторяющихся туров:**

1. Создайте базовый шаблон
2. Сохраните в каталоге
3. Используйте для новых смет
4. Корректируйте только даты и цены
