    Советы и трюки¶

    Версия: 2.3.0

    Эффективные способы работы, горячие клавиши и лучшие практики для Quote Calculator.

    🎯 Для кого этот раздел¶

    Новички: Научитесь работать быстрее с первых дней

    Опытные пользователи: Узнайте малоизвестные фишки и автоматизацию

    Профессионалы: Оптимальные workflow и интеграции

    📚 Содержание раздела¶

    Горячие клавиши¶

    Полный список всех keyboard shortcuts с примерами использования

    Что внутри: - Основные операции (Ctrl+S, Ctrl+N, Ctrl+O, Ctrl+P) - Работа с услугами (Ctrl+A, Delete, Escape) - Навигация (Tab, Shift+Tab, стрелки) - Bulk operations - Комбинации клавиш для эффективности

    Время изучения: 10 минут Экономия времени: 30% быстрее работа

    Эффективные Workflows¶

    Паттерны работы для типовых задач

    Что внутри: - Создание сметы за 2 минуты - Массовое повышение цен за 30 секунд - Организация каталогов - Шаблоны и пакеты - Интеграция с другими системами

    Время изучения: 20 минут Экономия времени: 50% быстрее типовые задачи

    ⚡ Топ-10 советов (Quick Start)¶

    1. Используйте пакеты услуг¶

    Вместо добавления услуг по одной:
Ctrl+K → Вкладка "Пакеты" → "Бали 7 дней"
→ 6 услуг добавлены за 1 клик

Экономия: 5 минут на смету

    2. Ctrl+A для массовых операций¶

    Повысить все цены на 10%:
Ctrl+A → "Цена" → "Умножить на 1.10"

Вместо: изменение 100 услуг вручную (30 мин)
С Ctrl+A: 30 секунд

    3. Дублирование смет (Ctrl+D)¶

    Похожая смета для нового клиента:
Открыть старую смету → Ctrl+D
→ Изменить имя клиента → Ctrl+S

Экономия: 10 минут создания с нуля

    4. Фильтры в каталоге¶

    Найти отели:
Каталог → Фильтр: "Категория = Accommodation"
→ Показываются только отели

Вместо прокрутки 500 услуг: моментальный поиск

    5. Backup перед критическими операциями¶

    Перед повышением цен или массовым удалением:
Ctrl+Shift+B → "Перед повышением цен"

Если что-то пойдёт не так:
Backups → Восстановить → 30 секунд

    6. Скрытая наценка для гибкости¶

    Вместо накрутки на каждую услугу:
Hidden Markup = 5%
→ Внутренняя маржа не видна клиенту
→ Легко изменить общую прибыль

Регулировка прибыли: 1 поле вместо 50

    7. JSON вместо CSV для кириллицы¶

    CSV с русскими названиями → проблемы с кодировкой
JSON → никогда проблем

Экспорт/импорт каталогов: только JSON

    8. Версионирование каталогов¶

    "Услуги Бали зима 2024-2025"
"Услуги Бали лето 2025"

Вместо перезаписи старого каталога:
Сохранить старый + создать новый
→ История цен доступна

    9. Keyboard shortcuts для скорости¶

    Выучить 5 основных:
Ctrl+N, Ctrl+S, Ctrl+K, Ctrl+P, Ctrl+A

Экономия: 20% времени на рутинных операциях

    10. Регулярные полные экспорты¶

    Каждую неделю:
Меню → Экспорт → Полный backup
→ Сохранить в Dropbox/Google Drive

Защита от потери данных: 100%
Время восстановления: 5 минут

    🚀 Повышение производительности¶

    Настройка рабочего пространства¶

    1. Dual monitor setup: 

    Монитор 1: Quote Calculator (смета)
Монитор 2: Email/документы клиента
→ Не переключаться между окнами

    2. Keyboard shortcuts на мышке: 

    Программируемая мышь:
Кнопка 4: Ctrl+S (сохранить)
Кнопка 5: Ctrl+K (каталог)
→ Сохранение одним кликом

    3. Browser bookmarks: 

    Закладка: http://localhost:4000
Горячая клавиша: Alt+1
→ Мгновенный запуск

    Автоматизация через API¶

    1. Ежедневный backup (cron): 

    #!/bin/bash
# Каждый день в 3:00
curl http://localhost:4000/api/export/all > backup_$(date +%Y%m%d).json

    2. Импорт прайсов поставщиков: 

    # Скрипт для автоимпорта CSV от отелей
import requests

with open('hotel_prices.csv') as f:
    prices = parse_csv(f)

for price in prices:
    requests.post('http://localhost:4000/api/catalogs/cat_001/services',
                  json=price)

    3. Генерация отчётов: 

    # Еженедельный отчёт по сметам
curl http://localhost:4000/api/stats > weekly_report.json

    💡 Малоизвестные фишки¶

    1. Bulk selection с фильтром¶

    Повысить цены только на отели на 15%:

1. Фильтр: Категория = Accommodation
2. Ctrl+A (выбрать все видимые)
3. "Цена" → "Умножить на 1.15"
4. Применить
5. Сбросить фильтр

→ Только отели изменены

    2. Копирование между каталогами¶

    Перенести услуги из одного каталога в другой:

1. Каталог А → Экспорт → catalog_a.json
2. Открыть в редакторе
3. Скопировать нужные services[]
4. Каталог Б → Импорт → вставить services
5. Сохранить

→ Дубликаты услуг в другом каталоге

    3. Быстрая корректировка цен¶

    Изменить цену на несколько рублей:

1. Кликнуть на цену услуги
2. Стрелка вверх/вниз: ±1
3. Ctrl+стрелка: ±10
4. Shift+стрелка: ±100

→ Точная настройка без ввода

    4. Drag-and-drop услуг¶

    Изменить порядок в смете:

Зажать услугу → перетащить вверх/вниз
→ Новый порядок (для группировки в печати)

    5. Multi-region каталоги¶

    Один и тот же каталог для разных стран:

"Стандартные услуги" + Region "Indonesia"
"Стандартные услуги" + Region "Thailand"
→ Разные цены, одинаковая структура
→ Легко сравнивать цены между странами

    Корпоративные стандарты¶

    Именование: 

    ✅ Сметы: "{Клиент} - {Направление} - {Даты}"
   Пример: "Иванов - Бали - 15-22 фев 2025"

✅ Каталоги: "{Направление} {Сезон}"
   Пример: "Бали зима 2024-2025"

✅ Пакеты: "{Направление} {Дней} дней {Уровень}"
   Пример: "Бали 7 дней стандарт"

    Backup policy: 

    Ежедневно: автоматические backups (система)
Еженедельно: полный экспорт (вручную)
Ежемесячно: архив на внешний носитель

    Управление каталогами: 

    Один человек отвечает за каталог
Изменения только через backup
Версионирование по сезонам

API скрипт: 5 секунд

    Простой backup: 30 секунд
