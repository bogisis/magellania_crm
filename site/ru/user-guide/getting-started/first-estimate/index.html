    Создание первой сметы¶

    Версия: 2.3.0 Дата обновления: 6 ноября 2025

    Введение¶

    Это пошаговое руководство покажет, как создать вашу первую смету в Quote Calculator. Весь процесс займет 5-10 минут.

    Что вы узнаете¶

    • ✅ Как запустить приложение
    • ✅ Как настроить параметры группы
    • ✅ Как добавить услуги
    • ✅ Как рассчитать стоимость
    • ✅ Как сохранить и распечатать смету

    Шаг 1: Запустить приложение¶

    Если установлено локально¶

    cd quote-calculator
npm start

    Если используется Docker¶

    docker-compose up -d

    Откройте браузер и перейдите по адресу: http://localhost:4000

    Вы должны увидеть главную страницу Quote Calculator:

    ![Главная страница]

    Шаг 2: Настроить информацию о клиенте¶

    В верхней части формы заполните данные клиента:

    Обязательные поля¶

    Имя клиента: 

    Иванов Петр Сергеевич

    Телефон: 

    +7 (999) 123-45-67

    Email: 

    ivanov@example.com

    💡 Совет: Email будет использоваться для идентификации сметы в системе.

    Шаг 3: Настроить параметры группы¶

    Количество человек (PAX)¶

    Укажите количество туристов в группе:

    PAX: 10

    💡 Важно: Многие услуги рассчитываются исходя из количества человек (трансферы, гиды, групповые экскурсии).

    Даты поездки¶

    Дата начала: 15.12.2025
Дата окончания: 22.12.2025

    Система автоматически рассчитает длительность: 7 дней / 6 ночей

    Шаг 4: Добавить услуги¶

    Есть два способа добавления услуг:

    Способ 1: Из каталога (рекомендуется)¶

    1. Нажмите кнопку "Открыть каталог" (Ctrl+O)
    2. Найдите нужную услугу в списке или используйте поиск
    3. Нажмите на услугу для добавления в смету

    Пример: Добавим трансфер

    Услуга: Трансфер аэропорт → отель
Категория: Трансферы 🚗
Цена: $100
Количество: 1

    Нажмите "Добавить"

    Способ 2: Создать custom услугу¶

    1. Нажмите кнопку "+ Добавить услугу"
    2. Заполните форму вручную:
    Название: Групповая экскурсия по городу
Цена: $350
Количество: 1
Наценка: 15%
    1. Нажмите "Добавить"

    Шаг 5: Добавим еще несколько услуг¶

    Давайте создадим полноценную смету для турпакета:

    Размещение¶

    Название: Отель 4* (двухместный номер)
Цена: $80 / ночь
Количество: 6 ночей
Итого: $480 × 5 номеров = $2,400
Наценка: 10%

    Питание¶

    Название: Завтраки в отеле
Цена: $15 / день
Количество: 7 дней
Итого: $15 × 10 человек × 7 дней = $1,050
Наценка: 5%

    Экскурсии¶

    1. Обзорная экскурсия по городу
   Цена: $350 (на группу)
   Количество: 1
   Наценка: 15%

2. Морская прогулка
   Цена: $500 (на группу)
   Количество: 1
   Наценка: 15%

    Гид-сопровождающий¶

    Название: Русскоговорящий гид (полный день)
Цена: $200 / день
Количество: 7 дней
Итого: $1,400
Наценка: 10%

    Шаг 6: Настроить наценки и налоги¶

    Индивидуальные наценки¶

    Каждая услуга может иметь свою наценку:

    Трансферы: 10%
Размещение: 10%
Питание: 5%
Экскурсии: 15%
Гид: 10%

    💡 Совет: Используйте разные наценки для разных категорий услуг в зависимости от рынка и конкуренции.

    Скрытая наценка¶

    Добавьте общую скрытую маржу на весь тур:

    Скрытая наценка: 5%

    ⚠️ Важно: Скрытая наценка НЕ показывается клиенту в печатной версии, но учитывается во внутренних расчётах.

    НДС (опционально)¶

    Если работаете с НДС:

    НДС: 0% (или ваша ставка)

    Шаг 7: Проверить расчёты¶

    Система автоматически рассчитывает:

    Базовая стоимость¶

    Трансферы:        $100
Размещение:       $2,400
Питание:          $1,050
Экскурсии:        $850
Гид:              $1,400
─────────────────────────
БАЗОВАЯ СТОИМОСТЬ: $5,800

    С наценками для клиента¶

    Трансферы (+ 10%):        $110
Размещение (+ 10%):       $2,640
Питание (+ 5%):           $1,102.50
Экскурсии (+ 15%):        $977.50
Гид (+ 10%):              $1,540
─────────────────────────
ИТОГО ДЛЯ КЛИЕНТА:        $6,370

    Ваша прибыль¶

    Индивидуальные наценки:   $570
Скрытая наценка (5%):     $290
─────────────────────────
ОБЩАЯ ПРИБЫЛЬ:            $860

    💰 Маржа: 14.8% от базовой стоимости

    Шаг 8: Сохранить смету¶

    Автосохранение¶

    Смета автоматически сохраняется каждые 30 секунд в localStorage браузера.

    Сохранить на сервер¶

    Нажмите кнопку "Сохранить" (Ctrl+S):

    ✅ Смета сохранена: ivanov_20251215_10pax_est_123.json

    Система создает: 1. Основной файл сметы в базе данных 2. Автоматический backup в таблице backups

    Экспортировать в JSON¶

    Для резервной копии или обмена:

    # Через интерфейс
Меню → Экспорт → JSON файл

# Через API
curl http://localhost:4000/api/estimates/est_123 > ivanov.json

    Шаг 9: Распечатать смету¶

    Подготовка к печати¶

    1. Проверьте все данные еще раз
    2. Убедитесь, что скрытые поля не показываются
    3. Нажмите "Печать" (Ctrl+P)

    Что попадет в печать¶

    Для клиента (видимые поля): - ✅ Информация о клиенте - ✅ Параметры группы (PAX, даты) - ✅ Список услуг с ценами - ✅ Индивидуальные наценки (показываются как часть цены) - ✅ Итоговая стоимость для клиента - ❌ Скрытая наценка (НЕ показывается) - ❌ Внутренние расчеты прибыли (НЕ показываются)

    Настройка печати¶

    В диалоге печати браузера:

    Ориентация: Портретная
Размер бумаги: A4
Масштаб: 100%
Поля: По умолчанию

    Сохранить как PDF¶

    Вместо печати выберите "Сохранить как PDF":

    Файл: ivanov_tour_2025-12-15.pdf

    💡 Совет: PDF можно отправить клиенту по email или мессенджеру.

    Шаг 10: Загрузить существующую смету¶

    Из списка на сервере¶

    1. Нажмите "Загрузить"
    2. Выберите смету из списка
    3. Нажмите "Открыть"

    Или используйте быстрый поиск:

    Ctrl+F → введите имя клиента → Enter

    Из JSON файла¶

    1. Нажмите "Импорт"
    2. Выберите JSON файл на компьютере
    3. Нажмите "Открыть"

    Система автоматически: - Загрузит все услуги - Восстановит параметры группы - Пересчитает итоги

    Шаг 11: Редактировать смету¶

    Изменить услугу¶

    1. Найдите услугу в списке
    2. Кликните на поле (цена, количество, наценка)
    3. Введите новое значение
    4. Нажмите Enter или кликните вне поля

    Система автоматически пересчитает итоги.

    Удалить услугу¶

    1. Наведите на услугу
    2. Нажмите кнопку "Удалить" (🗑️)
    3. Подтвердите удаление

    Или используйте клавишу Delete после выбора услуги.

    Изменить порядок услуг¶

    Перетащите услугу мышью на новое место (drag & drop).

    Типичные сценарии¶

    Сценарий 1: Быстрая смета на стандартный тур¶

    1. Открыть приложение
2. Заполнить данные клиента
3. Указать PAX и даты
4. Выбрать готовый пакет из каталога
5. Настроить наценки
6. Сохранить и распечатать

    Время: 2-3 минуты

    Сценарий 2: Индивидуальный тур с нуля¶

    1. Открыть приложение
2. Заполнить данные клиента
3. Указать PAX и даты
4. Добавить услуги вручную по запросу клиента
5. Настроить индивидуальные наценки
6. Добавить скрытую маржу
7. Сохранить и отправить клиенту PDF

    Время: 5-10 минут

    Сценарий 3: Редактирование существующей сметы¶

    1. Загрузить смету из списка
2. Изменить нужные параметры
3. Добавить/удалить услуги
4. Пересчитать
5. Сохранить под новым именем или перезаписать

    Время: 1-2 минуты
