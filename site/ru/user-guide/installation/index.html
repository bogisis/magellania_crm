    Установка Quote Calculator¶

    Версия: 2.3.0 Дата обновления: 6 ноября 2025

    Введение¶

    Quote Calculator v2.3.0 - это система для создания коммерческих предложений в туристическом бизнесе. Система может быть установлена несколькими способами в зависимости от ваших потребностей.

    Варианты установки¶

    🚀 Быстрый старт (рекомендуется)¶

    Для большинства пользователей рекомендуется локальная установка - простой и быстрый способ начать работу:

    # 1. Скачать проект
git clone https://github.com/your-org/quote-calculator.git
cd quote-calculator

# 2. Установить зависимости
npm install

# 3. Запустить сервер
npm start

# 4. Открыть в браузере
open http://localhost:4000

    Готово! Система запущена и готова к работе.

    👉 Подробная инструкция по локальной установке

    🐳 Docker (для продакшена)¶

    Если вам нужна продакшн установка с изоляцией и легким управлением:

    # Быстрый запуск с Docker
docker-compose up -d

# Открыть в браузере
open http://localhost:4000

    Преимущества: - ✅ Изолированная среда - ✅ Легкое обновление - ✅ Готовая конфигурация для продакшена - ✅ Автоматические backups

    👉 Подробная инструкция по Docker установке

    Системные требования¶

    Перед установкой убедитесь, что ваша система соответствует требованиям:

    • Node.js: 16.x или выше
    • npm: 8.x или выше
    • Память: Минимум 512MB RAM
    • Дисковое пространство: Минимум 100MB свободного места
    • Браузер: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+

    👉 Полные системные требования

    Выбор способа установки¶

    Когда использовать локальную установку?¶

    ✅ Подходит для: - Разработки и тестирования - Локальной работы на одном компьютере - Быстрого старта и экспериментов - Обучения и ознакомления

    ❌ Не подходит для: - Продакшн окружения - Многопользовательского доступа - Удаленного доступа через интернет

    Когда использовать Docker?¶

    ✅ Подходит для: - Продакшн окружения - Развертывания на сервере - Изолированной среды - CI/CD автоматизации - Легкого масштабирования

    ❌ Не подходит для: - Быстрого локального тестирования (избыточно) - Систем без Docker

    Сравнение способов установки¶

    Характеристика Локальная установка Docker
    Скорость установки ⚡ Быстро (5 мин) 🐌 Медленнее (10-15 мин)
    Простота ✅ Очень просто ⚠️ Требует знания Docker
    Изоляция ❌ Нет ✅ Да
    Продакшн ❌ Не рекомендуется ✅ Рекомендуется
    Обновление ⚠️ Вручную ✅ Автоматически
    Backups ⚠️ Вручную ✅ Автоматически
    Масштабирование ❌ Сложно ✅ Легко

    Быстрая диагностика¶

    После установки проверьте работоспособность системы:

    1. Health Check¶

    curl http://localhost:4000/health

    Ожидаемый ответ: 

    {
  "status": "healthy",
  "version": "2.3.0",
  "storage": {
    "type": "sqlite",
    "health": { "healthy": true }
  }
}

    2. Проверка интерфейса¶

    Откройте в браузере: http://localhost:4000

    Вы должны увидеть главную страницу Quote Calculator с формой создания сметы.

    3. Проверка API¶

    # Получить список смет
curl http://localhost:4000/api/estimates

# Получить список каталогов
curl http://localhost:4000/api/catalog/list

    Обновление¶

    Локальная установка¶

    # Сохранить данные
cp -r estimate/ backup/
cp -r catalog/ backup/

# Обновить код
git pull origin main
npm install

# Перезапустить сервер
npm start

    Docker установка¶

    # Обновить образы
docker-compose pull

# Перезапустить контейнеры
docker-compose up -d

    Удаление¶

    Локальная установка¶

    # Сохранить данные (если нужно)
cp -r estimate/ ~/backup/
cp -r catalog/ ~/backup/

# Удалить проект
rm -rf quote-calculator/

    Docker установка¶

    # Остановить контейнеры
docker-compose down

# Удалить volumes (ВНИМАНИЕ: удалит все данные!)
docker-compose down -v

# Удалить образы
docker rmi quote-calculator-app
