    Системные требования¶

    Версия: 2.3.0 Дата обновления: 6 ноября 2025

    Обзор¶

    Quote Calculator v2.3.0 - это легковесное веб-приложение с минимальными системными требованиями. Работает на большинстве современных систем.

    Минимальные требования¶

    Серверная часть (Backend)¶

    Компонент Минимум Рекомендуется
    CPU 1 core 2+ cores
    RAM 512 MB 1 GB
    Дисковое пространство 100 MB 500 MB
    Node.js 16.0.0 18.x LTS
    npm 8.0.0 9.x
    OS Linux, macOS, Windows 10+ Linux Server

    Клиентская часть (Frontend)¶

    Браузер Минимальная версия
    Google Chrome 90+
    Mozilla Firefox 88+
    Safari 14+
    Microsoft Edge 90+

    Не поддерживается: Internet Explorer

    Node.js¶

    Установка Node.js¶

    macOS (Homebrew): 

    # Установить последнюю LTS версию
brew install node@18

# Проверить версию
node --version
npm --version

    Linux (Ubuntu/Debian): 

    # Добавить репозиторий NodeSource
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -

# Установить Node.js
sudo apt-get install -y nodejs

# Проверить версию
node --version
npm --version

    Windows: 1. Скачать установщик с https://nodejs.org/ 2. Выбрать LTS версию (18.x) 3. Запустить установщик 4. Проверить установку в командной строке: 

    node --version
npm --version

    Альтернатива: nvm (Node Version Manager)¶

    Рекомендуется для разработчиков:

    macOS/Linux: 

    # Установить nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Перезагрузить terminal, затем:
nvm install 18
nvm use 18
nvm alias default 18

# Проверить
node --version

    Windows: 

    # Скачать nvm-windows
https://github.com/coreybutler/nvm-windows/releases

# После установки:
nvm install 18
nvm use 18

    macOS: 

    vm_stat | perl -ne '/page size of (\d+)/ and $size=$1; /Pages\s+([^:]+)[^\d]+(\d+)/ and printf("%-16s % 16.2f Mi\n", "$1:", $2 * $size / 1048576);'

    Linux: 

    free -h

    Windows: 

    Get-CimInstance Win32_OperatingSystem | Select-Object FreePhysicalMemory

    Процессор (CPU)¶

    Требования¶

    Минимум: 1 CPU core Рекомендуется: 2+ CPU cores Продакшн: 4+ CPU cores для нагруженных систем

    Проверка CPU¶

    macOS/Linux: 

    # Количество ядер
nproc

# Детальная информация
lscpu  # Linux
sysctl -n hw.ncpu  # macOS

    Windows: 

    Get-WmiObject Win32_Processor | Select-Object NumberOfCores,NumberOfLogicalProcessors

    Сеть¶

    Порты¶

    По умолчанию используется порт 4000.

    Убедитесь, что порт свободен:

    macOS/Linux: 

    lsof -i :4000

    Windows: 

    netstat -ano | findstr :4000

    Если порт занят, измените его через переменную окружения: 

    PORT=3000 npm start

    Требования к сети¶

    • Локальная работа: нет требований к интернету
    • Docker installation: требуется интернет для скачивания образов (~200 MB)
    • npm install: требуется интернет для скачивания пакетов (~100 MB)

    Firewall¶

    Если используете firewall, разрешите входящие подключения на порт 4000:

    Linux (ufw): 

    sudo ufw allow 4000/tcp

    macOS (pf): 

    # Добавить в /etc/pf.conf
pass in proto tcp to port 4000

    Windows Firewall: 

    New-NetFirewallRule -DisplayName "Quote Calculator" -Direction Inbound -LocalPort 4000 -Protocol TCP -Action Allow

    Проверка готовности системы¶

    Используйте этот чеклист перед установкой:

    #!/bin/bash
# check-requirements.sh

echo "=== Проверка системных требований ==="
echo ""

# Node.js
if command -v node &> /dev/null; then
    NODE_VERSION=$(node --version)
    echo "✅ Node.js: $NODE_VERSION"
else
    echo "❌ Node.js не установлен"
fi

# npm
if command -v npm &> /dev/null; then
    NPM_VERSION=$(npm --version)
    echo "✅ npm: v$NPM_VERSION"
else
    echo "❌ npm не установлен"
fi

# Git
if command -v git &> /dev/null; then
    GIT_VERSION=$(git --version)
    echo "✅ $GIT_VERSION"
else
    echo "⚠️  Git не установлен (опционально)"
fi

# Docker (опционально)
if command -v docker &> /dev/null; then
    DOCKER_VERSION=$(docker --version)
    echo "✅ $DOCKER_VERSION"
else
    echo "⚠️  Docker не установлен (опционально)"
fi

# Свободное место
echo ""
echo "Свободное место на диске:"
df -h . | tail -1

# Свободная память
echo ""
echo "Свободная память:"
free -h | grep Mem

echo ""
echo "=== Проверка завершена ==="

    Запустите: 

    chmod +x check-requirements.sh
./check-requirements.sh

    FAQ¶

    Нужен ли мне Docker?¶

    Нет, Docker опционален. Локальная установка проще для начала.

    Какую версию Node.js использовать?¶

    Рекомендуется Node.js 18.x LTS - стабильная версия с долгосрочной поддержкой.

    Можно ли запустить на Raspberry Pi?¶

    Да, но с ограничениями. Рекомендуется модель 4 с 4GB RAM.

    Работает ли на Windows?¶

    Да, Quote Calculator полностью поддерживает Windows 10 и 11.

    Нужен ли PostgreSQL/MySQL?¶

    Нет, Quote Calculator использует встроенный SQLite. PostgreSQL/MySQL поддержка планируется в будущих версиях.

    Сколько пользователей поддерживается?¶

    Один пользователь на инстанс. Для multi-user нужна дополнительная настройка.
