    👥 Навигация по ролям¶

    Я - Новый пользователь¶

    Путь обучения (1 час): 1. Installation → Установка 2. Getting Started → Основы интерфейса 3. First Estimate → Первая смета 4. Interface Guide → Детали UI 5. Printing → Как распечатать

    Часто нужно: - Keyboard Shortcuts - горячие клавиши - FAQ - частые вопросы

    Я - Опытный пользователь¶

    Эффективность: - Bulk Operations - массовое редактирование - Templates & Packages - готовые туры - Workflows - паттерны работы - Advanced Calculations - детали расчётов

    Управление данными: - Catalog Management - управление каталогами - Import/Export - обмен данными - Backup/Restore - резервное копирование

    Я - Разработчик (Frontend/Backend)¶

    Начало работы: - Architecture Overview - общая архитектура - Data Flow - поток данных - Business Logic - бизнес-логика

    API Reference: - All Endpoints - все эндпоинты - Estimates API - CRUD смет - Catalogs API - каталоги - Backups API - backups - Export/Import API - экспорт/импорт

    Development: - Testing - тестирование - Development Workflow - рабочий процесс (планируется)

    Я - DevOps/Администратор¶

    Deployment: - Deployment Overview - обзор развёртывания - Docker - Docker Compose - Production - production setup - Monitoring - мониторинг

    Maintenance: - Database - SQLite схема - System API - health checks

    📚 Навигация по темам¶

    Estimates (Сметы)¶

    User Flow: - Creating - создание - Editing - редактирование - Calculations - расчёты - Printing - печать

    API: - Estimates API - CRUD операции - Export/Import API - экспорт/импорт

    Architecture: - Business Logic - формулы расчётов - Data Flow - поток данных

    Catalogs (Каталоги)¶

    User Flow: - Overview - обзор каталогов - Managing - управление - Import/Export - импорт/экспорт

    API: - Catalogs API - CRUD, multi-region

    Architecture: - Database Schema - таблица catalogs

    Backups (Резервное копирование)¶

    User Flow: - Backup/Restore - создание/восстановление

    API: - Backups API - point-in-time recovery

    Architecture: - Database Schema - таблица backups - Data Flow - автосохранение

    Bulk Operations (Массовые операции)¶

    User Flow: - Bulk Operations - выбор, изменение, удаление

    API: - Batch API - batch update (планируется)

    Templates & Packages (Шаблоны и пакеты)¶

    User Flow: - Templates - создание и использование

    Architecture: - Data Structure - структура пакетов

    🔍 Cross-References (Перекрёстные ссылки)¶

    Estimates → API¶

    User Action API Endpoint Documentation
    Create estimate POST /api/estimates Estimates API
    Load estimate GET /api/estimates/:id Estimates API
    Update estimate PUT /api/estimates/:id Estimates API
    Delete estimate DELETE /api/estimates/:id Estimates API
    Export estimate GET /api/estimates/:id → JSON Export API

    Catalogs → API¶

    User Action API Endpoint Documentation
    Import catalog POST /api/catalogs Catalogs API
    Load catalog GET /api/catalogs/:id Catalogs API
    Update services PUT /api/catalogs/:id Catalogs API
    Export catalog GET /api/catalogs/:id → JSON Import/Export

    Backups → API¶

    User Action API Endpoint Documentation
    Create backup POST /api/backups Backups API
    List backups GET /api/estimates/:id/backups Backups API
    Restore backup POST /api/backups/:id/restore Backups API

    Bulk Operations → Implementation¶

    Operation Frontend Backend Documentation
    Select services selectedServices Set N/A Bulk Ops
    Change prices updatePrices() PUT /api/catalogs/:id Bulk Ops
    Delete services deleteSelected() PUT /api/catalogs/:id Bulk Ops
